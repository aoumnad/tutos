Liste des Commandes LaTeX Courantes

Encadré

\boxed{ }

Espacements

  • \, : petit espace
  • \: : espace moyen
  • \; : grand espace
  • \quad : espace d'une longueur de 1 "em"
  • \qquad : espace d'une longueur de 2 "em"

Commandes pour symboles mathématiques

  • \times : multiplication (×)
  • \div : division (÷)
  • \pm : plus ou moins (±)
  • \cdot : produit scalaire (⋅)
  • \infty : infini (∞)
  • \forall : pour tout (∀)
  • \exists : il existe (∃)
  • \sum : somme (∑)
  • \prod : produit (∏)
  • \int : intégrale (∫)
  • \cup : union (∪)
  • \cap : intersection (∩)
  • \Rightarrow : flèche double à droite (⇒)

Fractions et puissances

  • \frac{num}{den} : fraction (numérateur/dénominateur)
  • a^b : exposant (ab)
  • a_b : indice (ab)
  • a \over b : fraction simplifiée (a / b)

Racines

  • \sqrt{x} : racine carrée (√x)
  • \sqrt[n]{x} : racine nième (ⁿ√x)

Relations logiques et comparaisons

  • = : égal (=)
  • \neq : différent (≠)
  • \approx : approximativement égal (≈)
  • \lt : inférieur (\(\lt\))
  • \gt : supérieur (\(\gt\))
  • \leq : inférieur ou égal (≤)
  • \geq : supérieur ou égal (≥)
  • \gg : beaucoup plus grand que (≫)
  • \ll : beaucoup plus petit que (≪)
  • \in : appartient à (∈)
  • \notin : n'appartient pas à (∉)
  • \subset : sous-ensemble (⊂)
  • \supset : sur-ensemble (⊃)
  • \subseteq : sous-ensemble ou égal (⊆)
  • \supseteq : sur-ensemble ou égal (⊇)

Accents et symboles sur les lettres

  • \hat{x} : chapeau sur une lettre (x̂)
  • \bar{x} : barre sur une lettre (x̄)
  • \vec{x} : flèche sur une lettre (x⃗)
  • \dot{x} : point au-dessus (ẋ)
  • \ddot{x} : deux points au-dessus (ẍ)
  • \tilde{x} : tilde sur une lettre (x̃)

Parenthèses et délimiteurs

  • \left( ... \right) : parenthèses redimensionnables
  • \left[ ... \right] : crochets redimensionnables
  • \left\{ ... \right\} : accolades redimensionnables
  • |x| : valeur absolue (|x|)
  • \langle ... \rangle : angle (⟨ ... ⟩)

Divers

  • \boxed{} : \(\boxed{Encadrer}\)
  • \text{} : insérer du texte dans une formule
  • \lim : limite (lim)
  • \to : flèche vers (→)
  • \overline{} : barre supérieure (sur toute la longueur d'une expression)
  • \underline{} : souligner une expression
