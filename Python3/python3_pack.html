Le langage python 3 et l'Interface Graphique Tkinter
Le positionneur pack()
Le gestionnaire de géométrie pack() est plus simple et convient parfaitement au petites interfaces pas trop chargées.

Le gestionnaire pack() place les widgets dans l'espace qui n'est pas encore utilisé et adapte la taille de la fenêtre pour contenir tous les widgets.

Par défaut, les widgets sont placés les uns en dessous des autres en commençant par le haut (side=TOP). Pour contrôler le positionnement, on dispose des options suivantes:

  • L'option side, C'est l'option maîtresse du positionneur pack(). Elle permet de positionner le widget dans l'espace disponible. Elle peut prendre la valeur LEFT, RIGHT, TOP ou BOTOM. Au début, l'espace disponible est constitué de la totalité du widget père (conteneur). Une fois le widget positionné, on se retrouve avec un nouvel espace disponible dans lequel on recommence de la même manière.

 

 

 

  • l'option fill précise comment le widget va remplir l'espace disponible. Cette option peut prendre la valeur X pour horizontal, Y pour vertical, BOTH pour les deux et NONE pour aucun.
  • L'option expand permet préciser si le widget doit suivre le redimensionnement de son conteneur. cette option peut prendre la valeur 1 ou 0
  • Les options padx et pady permettent de définir les marges (en pixel) horizontales et verticales autour du widget (couleur marron)
  • L'option anchor. Par défaut, quand le widget ne remplit pas son espace, il est centré. Cette option permet de l'ancrer sur l'un des quatre cotés ou l'un des quatre coins.
  • Pour plus de souplesse, on peut commencer par placer des frames qui pourront à leur tour recevoir des widgets. A l'intérieur de ces nouveaux conteneurs, rien ne nous oblige d'utiliser le positionneur pack(), on peut très bien utiliser le positionneur grid() ou place()

 

Ce gestionnaire est très bien expliqué ici: http://www.gigante.be/python/didact_002.php

 

Exemple:

from tkinter import *
fp = Tk()
L1 = Label(fp, text = 'Nom')
E1 = Entry(fp, width=20)
L2 = Label(fp, text = '\nPrénom') #le \n permet de séparer un peu verticalement
E2 = Entry(fp, width=15)
B1 = Button(fp, text='Bouton1', bd=5)
B2 = Button(fp, text='Bouton2', bd=5)
L1.pack(anchor='w')
E1.pack()
L2.pack()
E2.pack(anchor='e')
B1.pack(pady=5)
B2.pack(pady=5, anchor='w')
fp.mainloop()

 Avec une petite modification on obtient:

from tkinter import *
fp = Tk()
L1 = Label(fp, text = 'Nom')
E1 = Entry(fp, width=20)
L2 = Label(fp, text = '\nPrénom') #le \n permet de séparer un peu verticalement
E2 = Entry(fp, width=15)
B1 = Button(fp, text='Bouton1', bd=5)
B2 = Button(fp, text='EXIT', bd=5, command=fp.destroy)
E3 = Text(fp, width=20, height=15)
E3.pack(side=RIGHT, fill=BOTH, padx=5, pady=10, expand=1)
L1.pack()
E1.pack()
L2.pack()
E2.pack()
B1.pack(pady=20)
B2.pack(pady=5)
fp.mainloop()
