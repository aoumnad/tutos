Le langage python 3 et l'Interface Graphique Tkinter
Le widget Entry
Ce widget d'une seule ligne  permet de saisir un texte. Pour récupérer le contenu du widget, on utilise la méthode nom.get(), qui retourne une chaîne. On peut utiliser les fonctions de conversion pour modifier le type.

 Voir cet exemple

Syntaxe:

from tkinter import *

nom = Entry(parent, option=valeur, option=valeur, . . .)


 Quelques options du widget Entry:

background (bg)
 		Couleur du widget "red", "blue", "green", "yellow", . . . ou #rrvvbb
borderwidth  (bd)
 		Epaisseur de la bordure en pixels. Défaut=2. améliore l'effet de relief
cursor
 		Curseur de la souris quand celle-ci survole le widget
disabledbackground
 		Couleur quand le widget est désacivé
disabledforeground
 		Couleur du texte quand le widget est désactivé
font
 		Fonte du texte du bouton. exemple: font="arial 8 italic bold"
foreground  (fg)
 		Couleur du texte
justify
 		Permet de justifier le texte (LEFT, CENTER, RIGHT)
relief
 		Relief du bouton.  Défaut = SUNKEN
show
 		Pour masquer les caractère tapés (mot de passe) show='*'par exemple
state
 		Permet d'activer (NORMAL) ou de désactiver (DISABLED) un bouton
 je n'ai pas bien compris le rôle de la valeur ACTIVE
validate
 		 
validatecommand
 		 
width
 		Largeur du bouton en caractère
 

 

 Quelques méthodes du widget Entry:

En plus des méthodes universelles , ce widget supporte les méthodes:


 .delete(first, [last])
 		Supprime des caractères de first jusqu'à last (non inclus)
 Edit1.delete(0,4)         supprime les caractères n° 0, 1, 2, 3
 Edit1.delete(4)         supprime le caractères n° 4   (le 5ème)
 Edit1.delete(0,END)       supprime tous les caractères du widget
.get()
 		Retourne le contenu du widget en tant que chaîne.
.insert(index, str)
 		Insère la chaîne str avant le caractère n° index
 

Les valeurs possibles des l'indices first, last, index:
0    indice du premier caractère
n    indice du (n+1)ème caractère
INSERT    position actuelle du curseur
END         fin du widget
ANCHOR    premier caractère de la zone sélectionnée (s'il y en a une)
