Syntaxe Python
Un principe fondamental du langage python est l'alignement (Indentation). Avec d'autre langage comme le langage C, on a l'habitude de décaler certains blocs de programme pour rendre le programme plus lisible. Avec Python, l'alignement des blocs de programme est devenu obligatoire. C'est la seule façon de délimiter blocs d'instruction. Par exemple:

 

Langage C:

En langage C, les deux programmes ci-dessous sont identiques. Celui de gauche est plus lisible, mais celui de droite fonctionne aussi car chaque bloc d'instructions est délimité par les accolades { }

main()
{
int i,j;
float k;
for(i=1; i < 100; i++){
                j = i * i;
                k = sqrt(i);
                printf("%d   %d    %f\n",i,j,k);
}
i = 1;
while(i < 100){
                j = pow(i,3);
                k = exp(i)
                printf("%d   %d    %f\n",i,j,k);
                i++;
}
}
 
main()
{
int i,j;
float k;
for(i=1; i < 100; i++)
{
j = i * i;
k = sqrt(i);
printf("%d   %d    %f\n",i,j,k);

while(i < 100)
{

Python:

Avec Python, il n'y a pas de délimiteur, le seul moyen de délimiter un bloc d'instructions est de  le décaler par rapport au reste. Les créateurs de ce langage, ont voulu ainsi forcer les programmeurs à travailler proprement. Le programme précédent devient.

          from math import *

for i in range(100):
j = i * i
k = sqrt(i)
print(i, j, k,'\n')
i = 1
while i < 100:
j = i ** 3
k = exp(i)
print(i, j, k,'\n')
i = i + 1

 

On remarquera que les instructions for et while se termine par (:) pour préciser qu'ils s'appliquent sur un bloque d'instruction qui doit avoir obligatoirement le même alignement.
