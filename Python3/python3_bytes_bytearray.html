Bytes  et  Bytearray
Les séquences de type bytes et bytearray  sont des suites d'octets. C'est un moyen efficace de représenter les chaines de caractère et les entiers (>255) par une suite d'octets (<=255)
les séquences bytes ne sont pas muttables, une fois définies, leurs éléments ne peuvent être modifiés.
les séquences bytearray sont muttables, leurs éléments peuvent être modifiés à n'importe quel moment.

La syntaxe pour définir une séquence de type byte ou bytearray est très similaire à celle d'une chaine de caractère. On utilise les simples guillemets ' ' ou les doubles guillemets  " " précédés de la lettre b, par exemple: bs = b'Bonjour' . Cette façon de définir une séquence bytes à partir d'une chaine n'est pas toujours utilisable. Elle marche seulement quand tous les caractères de la chaine peuvent être codés en ASCII (un octet par caractère)

Séquence de caractère: string Séquence d'octets : bytes
ss = 'ABCDE'
ss[0]  -->    'A'		 bs = b'ABCDE'
bs[0]  -->  65
ss = 'glaçon'
ss[3]   -->   'ç'		 bs = b'glaçon'  --> message d'erreur car le caractère ç ne peut être codé en code ASCII. Que faut il faire ?
On va utiliser la fonction de conversion bytes() en lui précisant le codage souhaité:

bs = bytes('glaçon', 'latin-1')  -->  b'gla\xe7on'
le caractère ç est codé par l'octet \xe7 = 231
bs[3] --> 231

bs = bytes('glaçon', 'utf-8')  -->  b'gla\xc3\xa7on'
le caractère ç est codé par 2 octets  \xc3  \xa7
bs[3] --> 195
bs[4] --> 167




Pour représenter un entier par une séquence de type bytes on utilise la méthode .to_bytes()

N = 65
Nb = N.to_bytes(1, 'big')   -->  b'A'
Nb[0]   -->  65

N = 250000
bin(N)  -->  '0b11 11010000 10010000'
Nb = N.to_bytes(3, 'big')  -->   b'\x03\xd0\x90'
Kb = N.to_bytes(3, 'little')  --> b'\x90\xd0\x03'
Nb[0]  -->  3
Nb[1]  -->  208
Nb[2]  -->  144
list(Nb)  -->  [3, 208, 144]
tuple(Nb)  -->  (3, 208, 144)


Pour former un entier à partir d'une séquence de type bytes, on utilise la classe:
int.from_bytes(bytes, byteorder, *, signed=False)

Bs = bytes([3, 4, 5])    -->    b'\x03\x04\x05'
N = int.from_bytes(Bs, 'big')    -->   197637
bin(N)    -->   '0b110000010000000101'  -->  11  00000100  00000101


Les objets bytes et bytearray supportent les opérations communes aux séquences en général ainsi des méthodes (à quelques exceptions près) s'appliquant aux séquences de type string.


Pour plus de détail, voir   http://docs.python.org/3/library/stdtypes.html#binaryseq
